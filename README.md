# Detolabot

    This is a telegram chat bot which promises to be interactive.    

    ![screenshot](/img/Telegrambot.png)



## Built With

- Ruby
- Telegram Bot API
- Chuck Norris Jokes API

## Feature

- The bot has two main features
- It gives you the random jokes based on food

## Commands to use

- /help
- /hello
- /start
- /bye
- /joke

## Dependencies
- telegram-bot-ruby
- dotenv
- httparty
